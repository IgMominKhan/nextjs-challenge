## Getting Started

First run the json server using the command

First, run the server:

```bash
npm run jsonserver
# or
yarn jsonserver
# or
pnpm jsonserver
# or
bun jsonserver
```

Then, run the server:

```bash
npm run start
# or
yarn start
# or
pnpm start
# or
bun start
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
